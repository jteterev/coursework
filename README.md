# Instapro

MVP социальной сети для обмена фотографиями

## Реализованная функциональность

### Основные задачи

- [x] Интеграция верстки списка постов с API
- [x] Реализация страницы добавления поста
- [x] Реализация страницы с постами конкретного пользователя
- [x] Реализация функционала лайков (ставить/снимать)
- [x] Динамическое отображение постов из API
- [x] Форматирование даты поста ("19 минут назад")
- [x] Обновление лайков без перезагрузки страницы

### Дополнительная функциональность

- [x] Обработка ошибок при создании поста
- [x] Обработка ошибок при постановке лайка
- [x] Отображение пустого состояния ленты
- [x] Валидация ввода при создании поста
